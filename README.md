# lcs-content-manager

## How to run

with cargo run

```bash
git clone https://github.com/koompi/lcs-content-manager
cargo run *path_to_settings.toml*
```

with binary

```bash
git clone https://github.com/koompi/lcs-content-manager
cd bin/
chmod +x lcs-content-manager
./lcs-content-manager *path_to_settings.toml*
```
